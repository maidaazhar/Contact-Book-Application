A Python-based Contact Book with input validation, simple encryption, efficient storage, file saving, and a console menu. Includes sample data for quick startup.

## Features
- Validate names, phone numbers, emails, and addresses; prevent duplicates.
- Encrypt emails and addresses using a simple Caesar Cipher.
- Add, view, search, update, delete, and sort contacts.
- Store contacts in a list (ordered) and a dictionary (fast lookup).
- Load and save contacts from `contacts.txt`; handles broken files.
- Console menu for easy navigation; optional sample data loader.

## Flow
1. Choose data source (file or sample data).
2. Load contacts into storage.
3. Use menu to manage contacts.
4. Automatically save changes on exit.

## Notes
- Designed for small datasets; great for learning and basic use.
- Combines list and dictionary for simplicity and performance.
- Validation and encryption enhance usability and privacy.
